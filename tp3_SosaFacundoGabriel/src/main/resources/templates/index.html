Modificadores de Acceso

"Public" "Private" "Protected" "Default"

Los modificadores de acceso o métodos accesores son palabras clave que se utilizan para controlar la visibilidad de clases, métodos y atributos en un programa.
Estos modificadores determinan desde qué partes del programa se puede acceder a una clase, método o atributo.

  1. Public
  2. Private
  3. Protected
  4. Default-package
Método Descripción
public Es accesible desde cualquier otro paquete. Esto significa que otras clases fuera del paquete en el que se encuentra la clase pueden acceder a ella.
private Solo es accesible dentro de la propia clase. No puede ser accesible desde ninguna otra clase, incluso si está en el mismo paquete
protected Permite que la clase sea accesible desde otras clases del mismo paquete y también desde clases que hereden de ella, incluso si están en paquetes diferentes
default Si no se especifica ningún modificador de acceso, la clase se considera tener un modificador de acceso "default" o "package-private". Esto significa que la clase es accesible solo dentro del mismo paquete en el que se encuentra

Clases

< Standar, Abstractas e Interfaces >

En Java, existen varios tipos de clases que pueden utilizarse para diferentes propósitos.
Algunos de los tipos de clases más comunes son:

  1. Standar
  2. Abstracta
  3. Interfaces
Tipo Descripción
Standar Son las clases que se utilizan para definir objetos y comportamientos en un programa. Pueden contener atributos, métodos y constructores para crear objetos y realizar operaciones.
Abstractas Son clases que no pueden ser instanciadas directamente y pueden contener métodos abstractos, es decir, métodos sin implementación. Se utilizan como plantillas para definir comportamientos que deben ser implementados por clases hijas.
Interfaces Son similares a las clases abstractas, pero solo pueden contener métodos abstractos y constantes. Se utilizan para definir contratos que las clases deben cumplir implementando los métodos definidos en la interfaz.

Clases por dentro

Constructores, atributos y métodos

Los constructores, atributos y métodos son elementos fundamentales dentro de las clases en Java, utilizados para definir el comportamiento y la funcionalidad de los objetos creados a partir de esas clases.

  1. Constructores
  2. Atributos
  3. Métodos
Elemento Descripción
Constructores Un constructor es un método especial que se llama automáticamente cuando se crea un objeto de una clase. Su principal función es inicializar el estado inicial del objeto, es decir, asignar valores iniciales a los atributos de la clase
Atributos Los atributos representan las características o propiedades de un objeto. Se definen dentro de la clase y especifican el tipo de datos y el nombre del atributo.
Métodos Los métodos son bloques de código que realizan tareas específicas o ejecutan acciones dentro de una clase. Pueden aceptar parámetros y devolver valores, o simplemente realizar una tarea sin devolver ningún valor.
